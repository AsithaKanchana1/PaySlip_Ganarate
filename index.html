📋 How to Use

  1. Place your Excel file (same format as Slary_Slips.xlsx) in the Excel/ folder.
  2. Set the Pay Period to the correct month and year.
  3. Click Generate Pay Slips PDF — or run the Python script directly.
  4. The output PaySlips_Output.pdf will appear in the project folder.
  5. Open the PDF and print — 6 slips per A4 page, ready to cut.
Quick run from terminal:
python3 generate_payslips.py
